## Description

The goal of this testing project is to ensure the proper functioning of a web-based shopping cart that allows 
users to add and remove products from a catalog.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
